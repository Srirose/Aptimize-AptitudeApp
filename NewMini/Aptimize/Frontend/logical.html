Letter and Symbol Analogy:

Involves finding relationships between symbols and letters. Common rules include:

Alphabet Position: Identify the position of a letter in the alphabet.

Example: A = 1, B = 2, C = 3, ..., Z = 26

Alphabetic Shift: Shift a letter forward or backward by a certain number to get another letter.

Example: If we shift A (1) by 3, we get D (4).

Opposite Letter: The opposite letter can be calculated by finding the letter at an equal distance from the end of the alphabet.

Example: A → Z, B → Y, C → X, etc.

Example with Shift:

If the given symbol is B (Position 2) and we apply a forward shift of 4, the new symbol will be F (Position 6).

Example with Opposite Letter:

If the given symbol is D (Position 4), its opposite letter will be W (Position 23) because it's 4 positions from the end of the alphabet.

Syllogism:

Syllogism is a logical argument where a conclusion is drawn from two given or assumed propositions (premises). Common forms of syllogism include:

Major Premise: A general statement or truth that is accepted.

Minor Premise: A specific statement related to the major premise.

Conclusion: A statement that logically follows from the premises.

Example of a Valid Syllogism:

Major Premise: All humans are mortal.

Minor Premise: Socrates is a human.

Conclusion: Socrates is mortal.

Common Forms of Syllogism:

  • All A are B, C is A, therefore C is B.
  • No A are B, C is A, therefore C is not B.
  • Some A are B, C is A, therefore C is B (or not B depending on the argument).

Valid vs Invalid Syllogism:

A syllogism is valid if the conclusion logically follows from the premises. If not, it is invalid.

Example of an Invalid Syllogism:

Major Premise: All dogs have four legs.

Minor Premise: A cat has four legs.

Conclusion: A cat is a dog. (Invalid conclusion; having four legs does not make a cat a dog)
